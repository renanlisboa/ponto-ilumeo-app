## Requerimentos

- [Node.js v16](https://nodejs.org/en/)


## Instalação

```bash
$ npm install
```

## Test

```bash
$ npm test
```

## Execução

```bash
ATENÇÃO! RENOMEAR ARQUIVO .env.local.example PARA .env.local E INSERIR URL DA API EM 'VITE_API_URL'

$ npm run dev
```

## Produção

```bash
$ npm run build
```
